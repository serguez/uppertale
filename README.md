# uppertale
Fan Game of Undertale

# Création de map : 
(0,0,255): mur
(0, 255, x): porte vers la map prochaine (renseignée dans config.py)
(255, 0, 0): joueur
(255, x, 255): ennemy (id = x)
(255, 255, x): pnj (id = x + 1)
